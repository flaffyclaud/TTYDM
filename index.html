TTYDM $
>_ Description: A Simple TTY Desktop Manager for Linux.
>_ Author: flaffyclaud
>_ Repository: github.com/flaffyclaud/TTYDM
>_ About: TTYDM is a simple TTY console Desktop Environment selector with a TUI interface coded using Shell Script.
Uses Gum for a rich TUI experience.
>_ Status: Early stages of development.
  • Retro green-on-black terminal style
  • Minimal documentation for easy onboarding
  • Open-source on GitHub
>_ Installation:
  • Install Gum: https://github.com/charmbracelet/gum
  • Copy the script to /usr/bin/ttydm and set appropriate permissions.
  • Modify the script according to your requirements.
>_ How to Use:
  • Run ttydm in your TTY session to select desktop environments via a TUI interface.
